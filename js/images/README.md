### 图片文件夹

- 建议就保持一个文件夹

  - drawable 其他图片 命名规则 按功能命名，类似 Android 的 drawable

- 自动生成图片文件

```
npm run images
```

在 js 文件中引用

```
import { empty01 } from '@/images';
```
