## 路由跳转

1. 跳转到下一路由

   - 不传参数
     msg.emit('router:goToNext', {
     routeName: 'XmArticle',
     });

   - 传参数
     msg.emit('router:goToNext', {
     routeName: 'XmArticle',
     data: "logistics"
     });

2. 返回上一页
   msg.emit('router:back');

3. 重置
   msg.emit('router:reset', {
   routeName: 'Tab',
   });
